# Visual Testing Skill

Use Browser DevTools MCP content, interaction, and Figma tools for visual testing.

## When to Use

- User wants to verify UI appearance or check for visual regressions
- User asks to compare page with design or Figma
- User needs responsive design testing (viewport resize)
- User mentions UI bugs, styling issues, or design parity

## Guidance

1. Use screenshot tools (region, full-page, format/quality options) for capture.
2. Use viewport resize tools to test different screen sizes (mobile, tablet, desktop).
3. Use ARIA snapshot then annotated screenshot for labeled element overlays when needed.
4. Use Figma compare tool when a Figma file key and node ID are available.
