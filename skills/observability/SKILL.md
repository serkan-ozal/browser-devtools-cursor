# Observability Skill

Use Browser DevTools MCP observability (o11y) tools for console, network, and trace correlation.

## When to Use

- User asks about distributed tracing or correlating frontend/backend requests
- User mentions OpenTelemetry, trace IDs, or monitoring
- User needs to debug request flow or inspect console/network behavior

## Guidance

1. Use trace ID tools to get, set, or create trace IDs for correlation.
2. Use console message tools to capture logs (filter by type or search text).
3. Use HTTP request tools to inspect requests, resource types, and status codes.
4. Use Web Vitals tools when performance metrics are also needed.
